# DBMS_LAB2
DBMS LAB ASSIGNMENT-2
College Database System

This assignment is based on designing a simple College Database System using ER modeling and SQL.

What I Did?
- Created an ER Diagram to illustrate entities and their relationships.
- Translated the ER model into relational tables.
- Implemented the tables using SQL DDL commands.
- Applied appropriate primary and foreign key constraints.
- Resolved the many-to-many relationship between Students and Courses using an Enrollment table.

Entities Used-

Department,
Student,
Faculty,
Course &
Enrollment
